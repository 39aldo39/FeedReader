**elementary OS / Ubuntu / Mint**

{: .text-black}
At least Gtk+ 3.12 required. So only elementary OS Freya or Ubuntu 15.04 or greater.

            add-apt-repository ppa:eviltwin1/feedreader-stable
            apt-get update
            apt-get install feedreader
